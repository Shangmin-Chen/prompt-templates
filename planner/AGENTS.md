# planner — executable engineering plan

Agent prompt that turns a high-level engineering instruction into a
concrete, ordered, self-sufficient plan (inside `<plan>` tags) for
another agent or developer to execute without follow-up questions.

The full prompt is in [`planner.md`](./planner.md). Referenced by the
one-shot Gemini/Cursor handoff protocol as the shape of `planner.md`
output.
