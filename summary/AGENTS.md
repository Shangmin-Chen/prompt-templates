# summary — codebase discovery output

Agent prompt template for read-only exploration: crawl a codebase and
produce a single `summary.md` that helps a human understand layout,
stack, entry points, architecture, and where to start reading.

The runnable prompt (scratchpad + `<prompt>` envelope) is in
[`summary.md`](./summary.md). Use it when a protocol or workflow needs
a structured codebase overview artifact.
